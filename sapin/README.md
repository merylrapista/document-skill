# AI-Powered Document Generation System 🤖📄

Welcome to the **Document Generation System**, a project developed during the **AI Productivity and Automation Day 2** workshop. This repository showcases how AI agents can leverage specialized skills to automate the creation of professionally formatted `.docx` documents. 

This project was brought to you by the **DICT Iloilo Provincial Field Office**.

## 🎯 Overview

This repository demonstrates the integration of AI agents with Python-based skills (`python-docx`) to streamline document generation. By combining the natural language processing capabilities of AI with programmatic document manipulation, we can automatically generate, format, and structure complex Word documents.

### Key Features
- **Automated `.docx` Generation**: Dynamically create Microsoft Word documents using Python.
- **AI Agent Integration**: Designed to be utilized by AI agents to process data and generate reports automatically.
- **Professional Formatting**: Utilize predefined templates, styles, and hierarchical structures for polished outputs.

## 🚀 Getting Started

### Prerequisites
- Python 3.x
- `python-docx` library

### Installation
Clone this repository and install the required dependencies:

```bash
git clone <repository-url>
cd docs_skill
pip install python-docx
```

## 🛠️ Usage

This project contains specific skills and scripts tailored for agentic workflows. For example, `generate_doc.py` uses the provided skills to generate custom `.docx` outputs based on inputs provided by the user or an AI agent.

```bash
python baroro/generate_doc.py
```
